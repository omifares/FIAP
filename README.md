# Arquivo Académico: Cibersegurança - FIAP

Este repositório funciona como o meu repositório central de material acadêmico. O objetivo é: consolidar, organizar e preservar os laboratórios, scripts e relatórios desenvolvidos ao longo da graduação em Cibersegurança na FIAP.

Mais do que apenas um repositório de código, esta é a minha biblioteca de referência. Aqui, encontro as soluções que desenhei para os desafios técnicos de cada semestre e os artefatos que suportaram a minha aprendizagem — desde scripts simples em Bash até análises forenses complexas.
Organização do Conhecimento

A estrutura reflete a cronologia do curso, facilitando a consulta rápida de conceitos ou ferramentas específicas:

 -  1o ano: A base. Aqui estão os fundamentos de redes, a transição para o ambiente Linux e os primeiros passos em automação com Bash e Python.
 -  2o ano: A fase de aprofundamento. Foco em análise forense, gestão de vulnerabilidades
 -  Global Solutions (GS), que documentam o meu desenvolvimento em desafios práticos mais robustos.

## Gestão Técnica

Para manter este arquivo funcional e organizado:

- Git LFS: Utilizado para gerir ficheiros volumosos (dumps de memória, capturas de rede, imagens de disco) que são indispensáveis para a reprodutibilidade dos laboratórios, mas que não se adequam ao versionamento padrão de código.
- Estrutura: Mantive o versionamento de forma modular, facilitando a busca de ficheiros específicos sem a necessidade de clonar o histórico completo, caso o foco de consulta seja apenas um semestre ou fase específica.
